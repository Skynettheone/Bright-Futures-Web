Team Page

Technical Description

The team page displays information about each team member using a flexible, responsive card layout. Each member is shown inside a styled card that includes an image, name, role, tasks, and social media links. Hover and focus effects reveal additional details, improving interactivity. The page is built with semantic HTML and modern CSS (Flexbox, shadows, transitions) to create an attractive and accessible presentation.

Accessibility

Accessibility was addressed on the team page by using semantic HTML elements, including headings and descriptive text for images, which help screen readers understand the content. Each team card includes a tabindex="0" attribute, allowing keyboard users to focus on cards and reveal details without a mouse. Clear link text and high-contrast colors further improve usability for all visitors.

Content Page

Technical Description

The content page presents information on sustainable energy topics such as electric vehicles, solar power, and wind power. It is structured using semantic HTML with clear headings and section IDs to enable internal linking and easy navigation. A sticky navigation menu and a "Go to Top" button improve user experience on long pages. The page uses responsive CSS for clean layout, styled images for visual support, and consistent design to ensure readability across devices.

Accessibility

Accessibility was addressed by using semantic HTML elements (such as headings and sections) to create a clear content structure for screen readers. Descriptive text was added to all images to ensure visually impaired users understand their purpose. The sticky navigation menu and internal anchor links help keyboard and assistive technology users easily navigate between sections. High-contrast colors and readable fonts further improve usability for all visitors.

Challenges and Lessons Learned
